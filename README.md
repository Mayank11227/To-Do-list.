# 📝 To-Do List Web Application

A simple and interactive **To-Do List Web Application** that helps users manage their daily tasks efficiently.  
This project allows users to add tasks, mark them as completed, and remove tasks when finished.

The goal of this project is to practice **JavaScript DOM manipulation, event handling, and basic front-end development skills.**

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📱 Responsive design
- ⚡ Fast and lightweight interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
